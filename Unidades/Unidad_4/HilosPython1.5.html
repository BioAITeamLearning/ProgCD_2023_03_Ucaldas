¬øC√≥mo utilizar los hilos?#

La forma m√°s sencilla de usar un subproceso es instanciarlo con una funci√≥n de destino y luego llamar al m√©todo start() para que comience a funcionar.

El subprocesamiento del m√≥dulo de Python tiene el m√©todo Thread():

class threading.Thread(group=None, target=None, name=None, args=(), kwargs={})

  • group: Este es el valor del grupo que debe ser Ninguno; esto est√° reservado para futuras implementaciones

  • target: esta es la funci√≥n que se ejecutar√° cuando inicie una actividad de subproceso

  • name: Este es el nombre del hilo; por defecto, se le asigna un nombre √∫nico de la forma Thread-N

  • args: esta es la tupla de argumentos que se pasar√°n a un destino

  • kwargs: este es el diccionario de argumentos de palabras clave que se utilizar√°n para la funci√≥n de destino

¬øC√≥mo implemento un hilo en una subclase?#

  • Definir una nueva subclase de la clase Thread

  • Llamar al constructor de la clase Thread es obligatorio; us√°ndolo, podemos redefinir algunas propiedades del hilo como el nombre del hilo.

  • Sobrecargue el m√©todo init_(self [,args]) para agregar argumentos adicionales

  • Luego, debo sobrecargar el m√©todo run(self [,args]) para implementar lo que debe hacer el subproceso cuando se inicia

  • En el programa principal, creamos varios objetos del tipo myThread; la ejecuci√≥n del hilo comienza cuando se llama al m√©todo start().

  • El subproceso se coloca en el estado activo de la llamada a start() y permanece all√≠ hasta que finaliza el m√©todo run() o lanza una excepci√≥n no controlada.

  • El programa finaliza cuando todos los subprocesos finalizan.

import threading
import time

class myThread (threading.Thread):
    def __init__(self, threadID, name, counter):
        threading.Thread.__init__(self)
        self.threadID = threadID
        self.name = name
        self.counter = counter
    def run(self):
        print ("Iniciando " + self.name)
        print_time(self.name, 5,self.counter)  
        print ("Finalizando " + self.name)
        
def print_time(threadName, delay, counter):
    while counter:
        time.sleep(delay)
        print ("%s: %s\n" %(threadName, time.strftime("%H:%M:%S", time.localtime(time.time())))) # Thread-#: HH:MM:SS
        counter -= 1

# Crear hilos
thread1 = myThread(1, "Thread-1", 1) # Objetos del tipo myThread # Hilo que muestra la hora cada 1 segundo
thread2 = myThread(2, "Thread-2", 4) # Objetos del tipo myThread # Hilo que muestra la hora cada 4 segundos

¬øHay que tomar precauciones al utilizar los hilos?#

  • Mientras el programa principal ha llegado al final, el hilo contin√∫a imprimiendo su mensaje cada dos segundos.

  • Este ejemplo demuestra qu√© son los subprocesos: una subtarea que hace algo en un proceso principal.

Un punto clave a tener en cuenta al usar subprocesos es que siempre debe asegurarse de nunca dejar ning√∫n subproceso ejecut√°ndose en segundo plano. Esta es una programaci√≥n muy mala y puede causarle todo tipo de problemas cuando trabaja en aplicaciones m√°s grandes.

from threading import Thread   
from time import sleep        

class CookBook(Thread):
    def __init__(self):
        Thread.__init__(self)
        self.message = "Hilo Trabajando!\n"
         
    def print_message(self):
        print (self.message)
         
    def run(self):
        print ("Hilo Inicia\n")
        x=0
        while (x < 3):
            self.print_message()
            sleep(2)  
            x += 1
        print ("Hilo Finaliza\n")    

print('\nPrograma principal inicia.\n')

# Librerias que utilizaremos
import threading   # Manejo de hilos
import numpy as np # Manejo de matrices
import time        # Manejo de tiempos

# Definir las 3 matrices de 4x10, con n√∫meros enteros entre 0 y 10
matrix1 = np.round(10*np.random.rand(4, 10)).astype(np.int8)
matrix2 = np.round(10*np.random.rand(4, 10)).astype(np.int8)
matrix3 = np.round(10*np.random.rand(4, 10)).astype(np.int8)

result_matrix = np.zeros((4, 10)).astype(np.int8)

    global result_matrix
    result_matrix[row_idx] = matrix1[row_idx] + matrix2[row_idx] + matrix3[row_idx]
    time.sleep(2) # Se demora 2 segundos sumando una fila

hilo3 = threading.Thread(target=sum_rows, args=(2,)) # Suma fila 3

hilo2.join()  # Esperar a que el hilo 2 termine

hilo3.join()  # Esperar a que el hilo 3 termine
